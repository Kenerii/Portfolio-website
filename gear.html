Gear

Objective: Creating a gear with Adobe Illustrator using a variety of different tools to give out a 3D perspective.

Problem: Using the appropriate amount of shadow and every part of the object to be symmetrical for the audience to have a real life perspective of the object.

Solution: The 3D effect was used on the object as well as the gradient for the edges. The pathfinder was used to give the width for the teeth.
