Halloween Poster

Objective: Creating a Halloween event poster through Adobe Illustrator using different types of tools to create a spooky effect.

Problem: Having to set the gradient at the correct amount on each specific part of the poster to give out a real life touch

Solution: Using the gradient on the bats to give the illusion that they are part of the brick wall. Having a gradient on each spider web strip as well as having it on the mask.
